# genscene

Make sure you are running Docker
Make sure you have python 3.12 installed (use pyenv)

In backend

- > docker compose up
- > python -m venv .venv
- > source .venv/bin/activate
- > pip install -r requirements.txt
- > python manage.py migrate
- > python manage.py runserver

In frontend

- > npm install
- > npm start
